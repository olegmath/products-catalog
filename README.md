# Planck School checkout

Статическая страница оплаты курса `Интенсив "Постойнное Повторение"` с интеграцией SOHO.LMS API.

## Куда добавить API-токен

Токен добавляется в файл `.env` в корне проекта:

```env
SOHO_API_TOKEN=ваш_токен_сюда
```

Файл уже создан здесь:

```text
/Users/hrabrovoleg/Documents/New project/.env
```

## Важно

Сейчас проект состоит из обычных файлов `index.html`, `app.js` и `styles.css`. Браузер не читает `.env` напрямую, поэтому текущий код ожидает токен в переменной:

```js
window.SOHO_API_TOKEN
```

Пока токен не передан в `window.SOHO_API_TOKEN`, приложение работает на мок-ответах и не делает реальные запросы к SOHO.LMS.

Не добавляйте `.env` в git: он уже внесён в `.gitignore`.

## Локальный запуск

Запустите локальный сервер:

```bash
npm run dev
```

Откройте страницу:

```text
http://localhost:4173
```

Локальный сервер читает `.env` и отдаёт токен в браузер через `/config.js`.
